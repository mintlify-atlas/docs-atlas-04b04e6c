## Built With Modern Web Technologies

This template uses industry-standard technologies for reliability and performance:

- **HTML5** - Semantic, accessible markup
- **CSS3 & SASS** - Modern styling with preprocessor support
- **JavaScript** - Interactive features and dynamic content
- **jQuery** - DOM manipulation and event handling
- **Google Forms** - Easy RSVP and data collection
- **Responsive Design** - Mobile-friendly across all devices

<Note>
  This template is based on the HTML5 UP Spectral theme and is available under the CC0-1.0 license, 
  allowing free use for personal and commercial projects.
</Note>

## Welcome to the Wedding Website Template

This is a feature-rich, responsive wedding website template designed to help couples create a beautiful online presence for their special day. Built with modern web technologies and based on the HTML5 UP Spectral theme, it provides everything you need to inform and engage your wedding guests.

## What Makes This Template Special

This template was created for a real wedding (Ankita & Anil, May 2022) and has been used successfully to manage hundreds of guests, coordinate logistics, and share the celebration with friends and family around the world.

## Technology Stack

This template is built with industry-standard web technologies:

- **HTML5** - Modern semantic markup with accessibility features
- **CSS3** - Responsive layouts with flexbox and media queries
- **SASS** - CSS preprocessor for maintainable stylesheets
- **JavaScript** - Interactive features and dynamic content
- **jQuery** - DOM manipulation and event handling
- **Font Awesome** - Beautiful icon library
- **Google Forms** - Easy form integration for RSVPs and data collection

<Note>
  The template uses vanilla JavaScript and jQuery, making it compatible with all modern browsers 
  without requiring a build process or complex dependencies.
</Note>

## What You Can Build

This template has been proven in real-world use and can handle:

- **Multi-day weddings** with different events (sangeet, ceremony, reception)
- **Multiple venues** with location maps and directions
- **Guest lists** of 300+ people with RSVP tracking
- **Photo sharing** with integration to external galleries
- **Live streaming** for remote guests during the pandemic era
- **Multilingual content** with proper charset support

## Based on HTML5 UP Spectral

This template uses the Spectral theme by HTML5 UP as its foundation, which provides:

- Clean, modern design aesthetics
- Responsive grid system
- Smooth scrolling animations
- Professional typography
- Well-organized SASS structure

The original theme has been customized and extended with wedding-specific features while maintaining the clean, professional look.

- ✅ Safari (latest 2 versions)
- ✅ Mobile Safari (iOS 12+)
- ✅ Chrome Mobile (Android 8+)

  Internet Explorer is not supported. The template uses modern JavaScript features (ES6+) 
  that are not available in IE11 and earlier.
</Warning>
